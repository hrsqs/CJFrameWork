# CJFrameWork (1.4.2)

1. WebService 服务器交互框架
2. AudioTool 音频处理框架
3. WebP 图像格式处理框架
4. AppStore信息获取
5. 异常捕获并提示

使用说明:
-----
1. 将顶部的构建工程改成</br>
![](https://github.com/hrsqs/CJFrameWork/raw/master/Screenshots/build.png)  
2. 点击Run生成FrameWork，不可使用快捷键Command+R
3. FrameWork文件生成在Products文件夹下，拷贝文件夹下的所有FrameWork文件至您的项目中
